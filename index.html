Analysis of Gradient Descent Variants for Neural Network Optimization

Overview

This project compares full-batch and mini-batch gradient descent variants for neural network training, focusing on convergence behavior and final MSE outcomes.

Current run architecture: a multilayer neural network with input dimension 5, two hidden layers (64 and 32 neurons), and one output neuron ([5, 64, 32, 1]).

Key Findings

  • Best method in the saved run: Momentum (Mini-Batch) with final test MSE 2.1752.
  • Mini-batch variants achieved lower final train and test MSE than full-batch variants in this run.

Methods Compared

  • SGD (Full Batch)
  • Mini-Batch GD
  • Momentum (Full Batch)
  • Momentum (Mini-Batch)
